# Marketing Website

The variscout.com marketing and documentation website.

---

## Overview

Built with Astro + React Islands:

- Static site generation for performance
- React islands for interactive demos
- MkDocs for documentation (this site)

---

## Technology Stack

| Component   | Technology      |
| ----------- | --------------- |
| Framework   | Astro           |
| Interactive | React Islands   |
| Styling     | Tailwind CSS    |
| Hosting     | Vercel          |
| Docs        | MkDocs Material |

---

## Site Structure

```
variscout.com
├── / (Homepage)
├── /journey (Interactive methodology demo)
├── /cases (Case study library)
├── /tools (Tool pages)
├── /learn (Educational content)
├── /products (Product pages)
├── /pricing (Pricing page)
├── /app (PWA application)
└── /docs (This documentation)
```

---

## User Flows

The website supports multiple user journeys:

1. **SEO Learner** - Google → Tool page → Product
2. **Social Discovery** - LinkedIn → Case → Product
3. **Content** - YouTube → Website → Product
4. **Enterprise** - Referral → Evaluation
5. **Return Visitor** - Direct → App

See [Journeys](../../02-journeys/index.md) for details.
